# Magneto Webstore

An open-source e-commerce webstore built with [Magento](https://magento.com/).

## Features

- Product catalog
- Shopping cart
- Checkout process
- User accounts
- Admin panel

## Installation

1. Clone the repository
2. Install dependencies
3. Configure database
4. Run setup
